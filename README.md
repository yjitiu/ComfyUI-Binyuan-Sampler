# ComfyUI-Binyuan-Sampler

一个为 ComfyUI 设计的全能采样器节点。

## 功能特点

- 一个节点整合完整工作流
- 支持 SD1.5 / SDXL / Flux / Z-Image 等模型
- LoRA 可视化开关
- 双上游图像输入
- 智能模型切换

## 安装方法

1. 进入 ComfyUI 的 `custom_nodes` 文件夹
2. 打开终端，运行：git clone https://github.com/yijitu/ComfyUI-Binyuan-Sampler.git
3. 重启 ComfyUI

V4.2
